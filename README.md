# 🎲 Random Number Generator

## 📌 Description

This is a simple Random Number Generator web application that generates random numbers within a specified range.
The project was built using HTML, CSS, and JavaScript to practice DOM manipulation and JavaScript functions.

## 🚀 Features

* Generates random numbers instantly
* User-friendly interface
* Uses JavaScript Math functions
* Responsive and simple UI
* Beginner-friendly project

## 🛠️ Technologies Used

* HTML5
* CSS3
* JavaScript (Vanilla JS)

## ▶️ How to Run the Project

1. Clone the repository:

```
git clone https://github.com/your-username/random-number-generator.git
```

2. Open the folder.

3. Double-click `index.html`
   OR
   Open `index.html` in your browser.
